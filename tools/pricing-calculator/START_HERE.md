# 🚀 START HERE

Welcome! This is the AWS Lambda Managed Instances Cost Calculator.

## Quick Deploy to GitHub Pages

### 1️⃣ Run the deployment script

```bash
cd CostCalculator_New
./PUSH_TO_GITHUB.sh
```

### 2️⃣ Enable GitHub Pages

1. Go to: https://github.com/debongithub/lmi-cost-calculator
2. Click **Settings** → **Pages**
3. Under "Build and deployment", select **GitHub Actions**
4. Save

### 3️⃣ Wait 2-3 minutes

Watch the deployment at:
https://github.com/debongithub/lmi-cost-calculator/actions

### 4️⃣ Visit your site

**https://debongithub.github.io/lmi-cost-calculator/**

## 🎯 What This Calculator Does

Compare costs across three deployment options:
- **Standard Lambda** - Pay-per-use serverless
- **Lambda Managed Instances** - Dedicated capacity with AWS management  
- **Self-Managed EC2** - Full control with manual management
